# Customer-Segmentation
A machine learning model that divides the customers into groups according to their expenditure.

## Pre-requisites
1. Python
2. Data Analsis libraries like Numpy,pandas.
3. Data Visualization libraries like Matplotlib,seaborn.
4. Scikit-Learn library for implementing ML algorithms.

## Technical Description
1. The whole source code is written in Python Programming Language.
2. It uses K-Means Clustering algorithm to divide the customers into groups.
3. Libraries used to analyse and clean the data are Numpy,Pandas,Matplotlib and Seaborn,
4. Scikit-Learn library is used to implement K-Means algorithm.
